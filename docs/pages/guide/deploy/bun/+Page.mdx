## Configuration

### Vite Configuration

Ensure that your Vite config includes the Photon Vite plugin:

```ts
// vite.config.ts
import { photon } from '@photonjs/runtime/vite'

export default {
  plugins: [
    photon(),
  ]
}
```

## Deployment

### Preview

Test build output locally with Node:

```bash
vite build
vite preview
```

### Production

Run build output directly:

```bash
vite build
bun --bun dist/server/index.mjs
```
